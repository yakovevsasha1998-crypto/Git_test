# Todo List API

Простое Todo-приложение с авторизацией на FastAPI и PostgreSQL.

## Технологии

- **FastAPI** — веб-фреймворк
- **PostgreSQL** — база данных
- **SQLAlchemy** — ORM
- **JWT** — авторизация
- **bcrypt** — хеширование паролей
